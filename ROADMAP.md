# Roadmap

## Next Up

- **Per-decision history diffs** — View what changed between versions of a decision, field by field
- **Per-project config** — `~/.decisionnode/projects/{name}/config.json` for project-specific threshold and agent behavior overrides

## Planned

- **VS Code extension** — View and manage decisions from the editor sidebar
- **Decision marketplace** — Browse and install community decision packs (e.g., "React best practices", "Go API conventions")

## Exploring

- **Cloud sync** — Optional E2E encrypted sync for teams
- **Team features** — Shared decision stores, role-based access
- **Decision templates** — Pre-built scopes and constraints for common frameworks
